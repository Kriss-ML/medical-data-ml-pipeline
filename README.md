# 🧠 Patient Hospitalization Prediction (Machine Learning)

End-to-end machine learning project for predicting patient hospitalization using simulated medical data.

---

## 🚀 Features

* Synthetic data generation with noise
* Data cleaning (duplicates, nulls, invalid values)
* Outlier detection using Local Outlier Factor (LOF)
* Feature scaling with StandardScaler
* Model training using Random Forest
* Model evaluation with multiple metrics
* Data visualization (ROC curve & performance metrics)

---

## 🧠 Skills Demonstrated

* Python for Data Science

* Data cleaning & preprocessing
* Outlier detection
* Machine Learning (Scikit-learn)
* Model evaluation (Accuracy, Precision, Recall, F1, ROC AUC)
